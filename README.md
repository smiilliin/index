# Index - Web Index

This is the index page of the web

## Install

```bash
npm install
```

.env

```
DB_USER=smile
DB_HOST=127.0.0.1
DB_PASSWORD=(Database password)
DB_DATABASE=(Database name)
RECAPTCHA_KEY=(Recaptcha key)
COOKIE_DOMAIN=(Cookie domain)
```

## Run

```bash
npm run build
npm run start
```
